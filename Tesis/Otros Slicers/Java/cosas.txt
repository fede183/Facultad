Javaslicer:

Necesitas un JAR, un método del cual iniciar, y una línea+variable que tomar como criterio. "Jolden completo.jar"

java -jar JoldenCompleto.jar 2 1000 
Programa 2, param 1000 (tiene muchos programas)

Instrumentás el programa y lo corrés a la vez generando una traza. Eso se hace con "tracer.jar"

java -noverify -javaagent:tracer.jar=tracefile:traza.trace -cp JoldenCompleto.jar main.Program 2 1000

Corrés el slicer, slicer.jar. con el criterio y la traza que generaste, genera un resultado. 
Largo, y no tan sencillo de leer

java -jar slicer.jar -p traza.trace bisort.BiSort.main:69:{sliceVariable} > slice.slice

Corrés mi programita "JavaslicerTranslator" que toma el resultado anterior y lo traduce en un formato de resultado, escrito IGUAL que el resultado de Slicer NET. esto sirve para comparar los resultados. 

./JavaslicerTranslator.exe slice.slice result.result trace.trace SummaryResultsFile.txt 1 1 1 BiSort 2 1000
